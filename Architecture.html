🔌 API Endpoints
MethodPathAuthDescription
POST/auth/register—Create new user account
POST/auth/login—Login, returns JWT token
GET/auth/meJWTGet current user profile
POST/documents/uploadJWTUpload file, triggers ingest job
GET/documentsJWTList all user documents + status
GET/documents/{id}JWTGet document status + metadata
DELETE/documents/{id}JWTDelete document + vectors
POST/chat/conversationsJWTCreate new conversation
GET/chat/conversationsJWTList conversations (pinned first)
POST/chat/sendJWTSend message, returns SSE stream
POST/competitorsJWTAdd competitor URL
GET/competitorsJWTList competitors + latest snapshot
POST/competitors/{id}/scrapeJWTTrigger manual re-scrape
POST/competitive/analyzeJWTStream competitive analysis (SSE)
POST/productsJWTCreate product definition
GET/productsJWTList products with linked docs
GET/health—System health (DB, Redis, Chroma)
